# Map-of-ecological-zones
## Приложение для нахождения маршрута миграции животных между экологическими зонами на основе работы с графами
Строится граф, где экологические зоны - вершины графа, экологические коридоры - ребра.
###  Используемые методы:
- Алгоритм обхода графа в ширину для поиска кратчайшего пути по количеству экологических коридоров
- Алгоритм обхода графа в глубину для проверки связности между экологическими зонами и обнаружения циклов
- Aлгоритм Дейкстры для нахождения кратчайшего пути между двумя экологическими зонами
### Также решаются такие задачи как:
- Нахождение точек сочленения графа
- Построение минимального остовного дерева
### Входные данные:
- Файл с описанием вершин графа (экологических зон) и ребрами графа (экологическими коридорами), представленными в виде списков смежности
### Как запустить через Visual Studio
- Склонируйте репозиторий 
- Откройте файл Lab_4-6.var26.sln
- Запустите программу
### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. Ссылка: https://github.com/MashaTitova/Map-of-ecological-zones.git
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования следует указать её название: git clone <ссылка на репозиторий> <папка>
