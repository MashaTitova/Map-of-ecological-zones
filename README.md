# Map-of-ecological-zones
## Приложение для нахождения маршрута миграции животных между экологическими зонами на основе работы с графами
Строится граф, где экологические зоны - вершины графа, экологические коридоры - ребра
###  Используемые методы:
- Алгоритм обхода графа в ширину для поиска кратчайшего пути по количеству эеологических коридоров
- Алгоритм обхода графа в глубину для проверки связности между экологическими зонами и обнаружения циклов
- Aлгоритм Дейкстры для нахождения кратчайшего пути между двумя экологическими зонами
### Также решаются такие задачи как:
- Нахождение точек сочленения графа
- Построение минимального остовного дерева
### Входные данные:
- Файл с описанием вершин графа (экологических зон) и списком
